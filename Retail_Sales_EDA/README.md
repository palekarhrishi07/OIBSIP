# Retail Sales EDA

## Objective
To perform Exploratory Data Analysis (EDA) on a retail sales dataset by cleaning the data, identifying patterns, and extracting meaningful insights.

## Steps Performed
1. Loaded the raw retail dataset.
2. Cleaned the data (handled missing values, removed duplicates, standardized column names).
3. Explored the dataset structure and performed descriptive statistics.
4. Analyzed customer demographics, product categories, and spending patterns.
5. Visualized sales distribution, gender-based revenue, product category performance, and monthly revenue trends.
6. Generated insights about customer behavior and sales trends.
7. Saved the cleaned dataset as `retail_cleaned.csv`.

## Files Included
- retail_sales_eda.ipynb  
- retail_cleaned.csv

## Output
A fully cleaned dataset and a complete EDA report with visualizations and insights useful for business decision-making.
